Introduction


워드프레스는 아름다움, 웹표준, 그리고 사용성에 중점을 둔 최신 기술의 시맨틱 개인 출판 플랫폼입니다.

워드프레스는 무료이지만 그 가치는 무한합니다.

워드프레스는 블로깅 소프트웨어로 작업하고자 할 때 쉽게 사용할 수 있는 도구이지 불편을 주는 도구가 아닙니다.

attach_money

무료

설치형 워드프레스는 무료로 제공되고 있습니다. 다양한 플러그인과 사용자 테마는 경우에 따라 유/무료로 제공됩니다.

web

웹표준

워드프레스는 사이트 접속자에게 별도의 프로그램 설치를 요구하지 않으며 모든 인터넷 환경에서 서비스가 가능하도록 웹표준을 준수하고 있습니다.

group

오픈소스

워드프레스는 오픈소스소프트웨어로 지속적인 업데이트와 개선이 이루어지고 있습니다. 다양한 디바이스 지원, 테마와 템플릿, 플러그인 제작 등 활발한 활동이 전세계에서 이루어지고 있습니다.

build

프레임워크

워드프레스의 웹사이트를 운영하기위한 핵심기능(Core), 콘텐츠관리시스템(CMS)을 간편한 사용환경과 함께 제공합니다.

dashboard

테마

테마는 웹사이트를 꾸밀 수 있게 도와주는 스킨 개념으로 전세계의 테마 제작자들이 다양한 테마를 제작해 제공합니다.

extension

플러그인

웹사이트에는 게시판, 통계, 보안 등 다양한 기능들이 필요한데 이런 웹사이트의 기능 및 기술을 모듈 단위로 만들어서 특정 웹사이트의 구축 목적이나 필요에 따라 연결할 수 있는 기능을 제공합니다.

More Information

In Korea?

2006년부터 한국 워드프레스 사용자 모임이 시작해 현재까지 다양한 활동을 하고 있습니다.

With github?

워드프레스 프로젝트는 Github에서 현재 활발하게 활동이 이루어지고 있습니다.

Writing Language?

워드프레스는 PHP로 작성되었습니다.

워드프레스는 MySQL 기반의 데이터베이스를 사용해 사용자의 업로드 데이터를 관리합니다.

.

How To Use

시작하기 앞서 아래의 내용이 갖추어져야 진행 가능합니다.

PHP 버전 5.6 이상

MySQL 버전 5.6 이상 또는 MariaDB 버전 10.0 이상

위 두 항목 구동 가능한 Apache 또는 Nginx 웹서버

1. DOWNLOAD

아래의 링크에서 워드프레스 최신버전을 다운로드해 주세요.

한글 버전 = KOR, 영문 버전 = ENG


2. Uncompression

다운로드 받은 파일의 압축을 풀어주세요.

압축을 푼 후 폴더명을 변경하는것이 보안을 강화하는데 좋습니다.

3. Open wp-config-sample.php

wp-config-sample.php 파일을 열어 자신의 환경에 맞게 변경합니다.

데이터베이스 이름
데이터베이스 사용자 이름
데이터베이스 비밀번호
데이터베이스 호스트
테이블 접두어

이상 5가지 항목을 미리 알고 있어야합니다.

사용 환경에 맞게 변경되었으면 파일명을 wp-config.php로 변경합니다.

4. File Upload

Filezila 또는 FTP 등을 통해 모든 파일을 호스팅하는 서버에 업로드 합니다.

5. Initial Setting

웹브라우저에 http://도메인/워드프레스폴더/wp-admin/install.php를 입력합니다.

이후 나타나는 안내에 따라 진행합니다.

6. Administrator Login

초기 설정을 마친 뒤 나타나는 화면에서 관리자 사용자명과 비밀번호를 입력합니다.

관리자 화면에 들어가서 콘텐츠를 입력해 사이트를 만들기 시작하면 됩니다.
