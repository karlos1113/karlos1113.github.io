  • Free Code Camp의 커리큘럼은 4가지로 나누어 진다. 위 증명서들은 표준화되었고, 고객이나 고용주에게 즉시 증명 가능할 만큼 효력이 있다. Free Code Camp의 다른 모든 것들과 마찬가지로 이 증명서들은 무료이다. 위 순서대로 차근차근 밟아나가는 것을 추천하지만 원한다면 건너 뛰어도 괜찮다. 각 증명은 400시간이 걸리고 마지막은 총 800시간이 소요된다.






  • Front End Development Certification을 얻기 위해서는 HTML, CSS, Jquery, Java script를 이용하여 10개의 프로젝트를 완수해야 한다.






  • Data Visualization Certification을 얻기 위해서는 React, Sass, D3.js를 이용하여 10개의 프로젝트를 완수해야한다.






  • Back End Development Certification을 얻기 위해서는 Node.js, Express, and MongoDB를 이용하여 10개의 프로젝트를 완수해야한다. 이것들을 클라우드에 올리기 위해서 Git과 Heroku를 사용할 것이다.






  • 세개의 증명서를 모두 완료한 후에, 다른 캠퍼와 팀을 이뤄서 agile software development 방법을 사용하여 두가지의 실생활의 프로젝트를 진행하게 된다. 그리고 legacy code에 functionality를 추가하게 된다. 프로젝트가 끝날 쯤에, 자신의 포트폴리오를 갖게 된다.






  • 2080시간의 모든 challenge들과 프로젝트를 완료하게 되면, Full Stack Devlopment Certification을 얻게 된다. 그러면 Free Code Camp에서 무료로 code 인터뷰 연습을 제공하고, 심지어 회사 고용주들에게 추천을 해준다.
