  • 코딩을 시작하기전에, 먼저 Free Code Camp 채팅방에 참여해보자. 언제든지 채팅방에 가서 사람들과 어울리거나 질문을 하거나 같이 코딩을 할 짝을 찾을 수 있다. 하지만 먼저 Github 계정이 필요하다.






  • Github 계정이 있다면, Github에 로그인과 동시에 채팅방에 참여할 수 있다. 먼저 "Hello world!"라고 자신을 소개해보자! 그리고 동료 친구들에게 어떻게 Free Code Camp에 가입하게 되었는지 말해보는 것도 좋다. 또한 왜 코딩을 시작하는 지도 말해보자.






  • 채팅방이 매우 활동적이기 때문에 채팅방 설정을 통해서 누군가 나를 언급했을 때만 알림을 주도록 하는 것이 좋다.
